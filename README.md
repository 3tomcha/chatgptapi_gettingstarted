# 未来日記
目標達成のために面倒なスケジュールを立てますか？  
Non!それはもう必要ありません。  
これからは、あなたは目標を入力するだけ。AIが自動的にスケジュールを立ててくれるから、その通りに行動するだけで目標が達成できちゃいます。  
それが「未来日記」です。さぁデザインされた未来へ飛び込もう！
